Welcome to CAT: The Princeton Course Aggregation Tool

We provide you with tools that make looking for courses powerful and painless:

  • Our smart search bar allows you to search for courses like you search for everything else, with Google-like queries.
  • To streamline your results, we hide courses from previous semesters by default. If you're looking for a course that might not be offered every semester, be sure to click the "View Previous Semesters" button next to the search bar. Courses from previous semesters will then show up as grey results.
  • Our smart search bar is still learning! If you're not getting the results you expect, give "Advanced Search" a try, next to the search bar.
  • Each search result displays a colored circle that represents the course's overall SCORE rating over time.
  • You can sort search results by subject/number, professor, or overall rating.
  • Once you find the search result you're looking for, one click on the result displays the Registrar's data in the same format you're used to reading on their site. Clicking "See Reviews" will bring you data from Princeton's required course evaluations, both graphs and written reviews.
  • You can also see data from past offerings of a course by using the "Select Semesters" dropdown menu.
  • If you want to save a course for future reference, click the cart icon in the results. We'll store it for you in your cart (at the bottom of the page) so you can find it quickly later, even if you log out.
